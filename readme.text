3. What is Event Bubbling? And how does it work?

ans:
Event Bubbling means when an event happens on child element,it first runs on that element then moves up step by step to its parenst,grandparents and so on.

4. What is Event Delegation in JavaScript? Why is it useful?

ans:
Event Delagation means putting one event listener on a parent element instead of many listeners on each child.
also it is useful for saving memory,less code,easier to manage dynamic element.
